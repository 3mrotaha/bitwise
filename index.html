Our Projects

RC Car using STM32

Remote-controlled car using STM32 Nucleo-F446RE (FreeRTOS) and ESP32, featuring MQTT/Bluetooth control, telemetry, and multi-sensor integration for advanced monitoring.

Bee's Venom Collector

A smart device that uses precise frequency generation to attract bees and collect venom efficiently, aiding research and medicinal use.

Smart Greenhouse

An automated greenhouse system powered by Raspberry Pi 5, monitoring plant growth, optimizing conditions, and offering remote control based on full client specifications.

ESP Flasher

A fast, reliable ESP32-based solution to store and flash binary files into other ESP32 boards without needing a computer connection.

Secure Door Notifier

Home security system using ESP32 and GSM SIM800L for real-time notifications via Telegram and SMS in case of unauthorized access.

NIR Glucometer

Developed a system for measuring glucose levels using Near-Infrared (NIR) sensors for non-invasive monitoring.

Connect-4 Console Game

Client/server console game with AI/player modes and difficulty levels, allowing two players to compete in connecting four cells to win.

GUI Quizlet

Desktop application presenting random questions from a database with server-side grading and multiple difficulty levels.

Digital People Counter

Hardware project using digital logic (gates, flip-flops, counters, sensors) to count people entering/leaving an area with alert capabilities.
